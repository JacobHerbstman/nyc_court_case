# DOB FOIL Request Draft

Submit through the NYC OpenRecords portal and select `Department of Buildings (DOB)`:

- Portal: <https://www.nyc.gov/openrecords>
- DOB FOIL page: <https://www.nyc.gov/site/buildings/dob/foil-requests.page>

Suggested title:

`Pre-2000 DOB New Building permit and job filing records (1980-1999)`

Suggested request text:

I request electronic copies of Department of Buildings records for New York City new building activity from January 1, 1980 through December 31, 1999.

Please provide any machine-readable database extracts, flat files, indexes, logs, ledgers, or other electronic record exports that capture new building permits, new building job filings, permit issuance, or closely related records maintained by DOB for that period.

My primary request is for records covering New Building (`NB`) jobs or their pre-digital / historical equivalent. If DOB did not use the modern `NB` label in part of this period, please provide the closest historical equivalent used to identify new building applications or permits for residential construction.

For each record, please provide the following fields where available:

- job, application, or filing number
- permit number
- job type or permit type
- filing date or application date
- approval date
- permit issuance date
- sign-off date, completion date, or certificate-of-occupancy-related date
- borough
- block
- lot
- BBL if stored directly
- BIN if available
- house number
- street name
- ZIP code if available
- existing and proposed use or occupancy
- existing and proposed dwelling-unit counts
- existing and proposed stories, floor area, or building class
- applicant, owner, or filing representative fields if available
- any status fields indicating approval, withdrawal, denial, completion, or expiration

If a single extract with all of the above is not available, please provide the closest available record set or sets that together document:

1. new building filings or applications,
2. approvals or permit issuance, and
3. completion or occupancy status.

If DOB maintains codebooks, record layouts, data dictionaries, field definitions, or documentation for these historical record systems, please provide those as well.

I prefer delivery in electronic machine-readable format such as CSV, TSV, pipe-delimited text, database export, or Excel workbook. If the underlying records exist only in scanned or microfilm form, please first provide any available electronic index, inventory, or ledger identifying the responsive records and their locations.

If this request is too broad, please contact me through NYC OpenRecords so I can narrow it. My preferred first priority is a borough-block-lot-address keyed extract of new building filings and permits for 1980-1999 with dates and dwelling-unit fields.

Notes for our project:

- We only need records, not explanations or narrative responses.
- The most important use is constructing a historical lot- or address-level panel of new residential construction activity before 2000.
- If DOB suggests another agency or archive for portions of these records, record that referral in `tasks/source_registry/code/archive_requests.csv`.
